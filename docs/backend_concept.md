# ミノデ不動産 バックヤードの仕組み

## コンセプト：電話1本で物件ページが完成

```
従来の方法                    ミノデ不動産の方法
─────────────────────────────────────────────────────
書類を見ながら               電話で話すだけ
パソコンで入力    →          ↓
写真をアップロード           AIが自動でまとめる
レイアウト調整               ↓
公開作業                     確認して公開
─────────────────────────────────────────────────────
作業時間: 数時間              作業時間: 15分
```

---

## 情報入力の方法（2つ）

### 方法1: 電話で話す

```
┌─────────────────────────────────────────────┐
│                                             │
│   細川さん・青木さん                         │
│        ↓                                    │
│   美濃出に電話                              │
│   「この物件なんだけど...」                  │
│        ↓                                    │
│   通話を録音                                │
│        ↓                                    │
│   AIが自動で文字起こし                       │
│        ↓                                    │
│   AIが物件情報を整理                         │
│   ・住所、間取り、築年数                     │
│   ・家賃、敷金礼金                          │
│   ・おすすめポイント                        │
│   ・注意事項                                │
│        ↓                                    │
│   物件ページ自動生成                         │
│        ↓                                    │
│   確認 → 公開                               │
│                                             │
└─────────────────────────────────────────────┘
```

**話す内容の例：**
> 「富士見駅から歩いて8分くらいの一戸建てがあってね。
> 4DKで家賃4万円。敷金礼金は4万ずつ。
> 築60年くらいだけど、トイレとお風呂はリフォームしてある。
> DIYしたい人にはいいと思うよ。
> あ、駐車場は敷地内にないから、近くで借りる必要があるね。」

これだけで物件ページの8割が完成します。

### 方法2: 書類をアップロード

```
┌─────────────────────────────────────────────┐
│                                             │
│   登記簿・図面・写真など                     │
│        ↓                                    │
│   スマホで写真を撮る                         │
│   または PDFをアップロード                   │
│        ↓                                    │
│   AIが自動で読み取り                         │
│   ・住所、面積、築年数                       │
│   ・間取り図から部屋情報                     │
│        ↓                                    │
│   物件ページ自動生成                         │
│                                             │
└─────────────────────────────────────────────┘
```

---

## データの流れ

```
入力                    AI処理                  出力
──────────────────────────────────────────────────────

電話の録音  ─┐
             ├──→  AIが整理  ──→  物件データ  ──→  物件ページ
書類の写真  ─┘                    （JSON形式）      （自動生成）

```

### 物件データの例

```
物件番号: 4654
住所: 長野県諏訪郡富士見町富士見4654-490
種別: 賃貸・一戸建て
間取り: 4DK + 物置
面積: 95.88㎡
築年数: 63年
構造: 木造平屋建て

賃料: 40,000円/月
敷金: 40,000円
礼金: 40,000円

最寄駅: 富士見駅 徒歩8分
駐車場: なし（近隣 月2,000円）

特徴:
- トイレ・お風呂リフォーム済み
- DIY歓迎
- ペット相談可

注意事項:
- 残置物は入居前に撤去
- 断熱性能は弱め
```

このデータがあれば、物件ページは自動で作れます。

## 60代以上の方にとってのメリット

| 従来 | ミノデ不動産 |
|------|-------------|
| パソコン操作が必要 | 電話で話すだけ |
| 入力項目を覚える | 知っていることを話す |
| 写真のアップロード | スマホで撮って送るだけ |
| レイアウト調整 | 自動でいい感じに |

**「パソコンは苦手だけど、物件のことは詳しい」**

その知識を、電話で話すだけで活かせます。

## 次のステップ

1. **試しに1件やってみる**
   - 細川さんの持っている物件を1つ選ぶ
   - 電話で情報を教えてもらう
   - 物件ページを作成して見せる

2. **農地版も検討**
   - 農地情報を地図にプロット
   - 「この辺りで農地を探している人」と
   - 「使ってほしい農地を持っている人」をつなげる
